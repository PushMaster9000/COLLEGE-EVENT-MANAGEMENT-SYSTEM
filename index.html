College Event Management System

A comprehensive database-driven platform for managing college events, participants, and resources with MySQL backend.

Database Schema Design

Our system uses a normalized relational database with multiple interconnected tables

Users Table

  • user_id INT (PK)
  • name VARCHAR(100)
  • email VARCHAR(100)
  • password VARCHAR(255)
  • role ENUM
  • department VARCHAR(50)

Events Table

  • event_id INT (PK)
  • title VARCHAR(255)
  • description TEXT
  • date DATE
  • time TIME
  • location VARCHAR(255)

Registrations Table

  • registration_id INT (PK)
  • user_id INT (FK)
  • event_id INT (FK)
  • registration_date TIMESTAMP
  • status ENUM
